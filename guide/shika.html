Инструкция по установке

  1. Registrasi Akun

    Buka & daftarkan akun baru Anda di shikahub.org

  2. Login

    Masuk menggunakan kredensial yang baru saja Anda buat.

  3. Aktivasi Kunci (Key)

    Input atau klaim kode lisensi yang sudah Anda beli di menu dashboard akun.

  4. Unduh Launcher

    Pergi ke halaman unduhan: shikahub.org/en/dashboard/downloads

    PENTING: Halaman unduhan tidak akan bisa diakses jika Anda melewati langkah registrasi dan aktivasi key di awal.
  5. Jalankan Loader

    Ekstrak file, klik kanan pada launcher, lalu pilih Run as administrator.

  6. Jalankan Shika

    Pilih dan eksekusi opsi Shika di dalam menu loader.

  7. Buka Game

    Jalankan game Anda. Sistem otomatis berjalan.

  1. Register Account

    Open and register a new account at shikahub.org

  2. Login

    Log in using the credentials you just created.

  3. Activate Key

    Input and redeem the license key you purchased in the account dashboard.

  4. Download Launcher

    Go to the downloads page: shikahub.org/en/dashboard/downloads

    NOTE: You cannot access the download link if you do not follow the steps from the very beginning (Register & Activate Key).
  5. Run as Admin

    Extract the file, right-click the downloaded launcher, and select Run as administrator.

  6. Run Shika

    Select and launch the Shika option inside the loader menu.

  7. Run the Game

    Start your game. The system will automatically inject.

  1. 注册账号

    打开并注册新账号： shikahub.org

  2. 登录系统

    使用刚才注册的账号信息登录网站。

  3. 激活卡密 (Key)

    在个人中心控制台输入并激活你购买的授权卡密。

  4. 下载启动器

    前往下载页面： shikahub.org/en/dashboard/downloads

    注意： 如果你不按照顺序进行（注册和激活卡密），你将无法访问该下载链接。
  5. 管理员运行

    解压文件，右键点击下载的启动器应用程序，选择 以管理员身份运行。

  6. 启动 Shika

    在加载器菜单中选择并运行 Shika 功能。

  7. 启动游戏

    打开并运行游戏，程序将自动生效。

  1. تسجيل الحساب

    افتح وسجل حسابًا جديدًا عبر الرابط: shikahub.org

  2. تسجيل الدخول

    قم بتسجيل الدخول باستخدام البيانات التي أنشأتها للتو.

  3. تفعيل المفتاح (Key)

    أدخل مفتاح الترخيص الذي اشتريته في لوحة التحكم لتفعيله.

  4. تحميل المشغل

    انتقل إلى صفحة التحميل: shikahub.org/en/dashboard/downloads

    ملاحظة مهمة: لن تتمكن من الوصول إلى صفحة التحميل إذا لم تتبع الخطوات من البداية (التسجيل والتفعيل).
  5. التشغيل كمسؤول

    فك الضغط عن الملف، انقر بزر الماوس الأيمن على تطبيق المشغل، ثم اختر **Run as administrator**.

  6. تشغيل Shika

    اختر وقم بتشغيل قائمة Shika من داخل برنامج اللودر.

  7. تشغيل اللعبة

    أخيرًا، افتح وشغّل اللعبة الخاصة بك. تم الإعداد بنجاح.

  1. Регистрация аккаунта

    Перейдите и зарегистрируйте новый аккаунт на сайте: shikahub.org

  2. Вход (Login)

    Войдите в систему, используя учетные данные, которые вы только что создали.

  3. Активация ключа

    Введите купленный лицензионный ключ в поле активации в личном кабинете.

  4. Скачивание лаунчера

    Перейдите на страницу загрузки: shikahub.org/en/dashboard/downloads

    ВАЖНО: Вы не сможете получить доступ к ссылке на скачивание, если не выполните шаги с самого начала (регистрация и активация ключа).
  5. Запуск от админа

    Распакуйте архив, нажмите правой кнопкой мыши на лаунчер и выберите Запуск от имени администратора.

  6. Запуск Shika

    Выберите и запустите Shika в меню открывшегося лоадера.

  7. Запуск игры

    Запустите игру. Система автоматически начнет работу.

Launcher FAQ WIP

HWID Mismatch Message
Coba reset HWID Anda langsung via panel website. Jika limit habis atau gagal, silakan buat tiket aduan di menu dashboard web.
Launcher Won't Start (Error 0xc067 / Sejenisnya)

• Disarankan menggunakan Windows 11 versi terbaru (25H2).

• Jika belum aktif, buka Windows PowerShell lalu jalankan perintah: irm get.activated.win | iex untuk aktivasi gratis.

• Jika masih crash, buka tiket di web sertakan screenshot error code agar tim dev bisa cek manual. Alternatif lain: Coba buka paksa terus-menerus.

Page Infinite Load (Situs Muter Terus)
Hal ini normal terjadi jika session login habis (auto log-out). Solusinya, kembali ke homepage utama shikahub.org atau klik logo Shika di ujung kiri atas untuk login ulang.
Proses Migrasi Akun (Migrating)
Pemberian role migrasi sepenuhnya diatur dev. Klik tombol migrasi di web, jika tertulis "Already Registered", Anda cukup memastikan akun Shikahub sudah link dengan akun Discord Anda. Sisa sub otomatis terkonversi setelah sinkronisasi beres.
Kode Verifikasi Email Tidak Masuk
Pastikan untuk memeriksa folder Spam atau Promosi pada email yang Anda daftarkan.
Autorisation Error: Empty Response from token endpoint
Hapus file konfigurasi usang cfg.ini pada direktori komputer Anda berikut:
%LOCALAPPDATA%/shika/launcher
HWID Mismatch Message
Try to reset your HWID over the website dashboard. If that doesn't work, please create a support ticket on the website!
Launcher Won't Start (Error 0xc067 or similar)

• You can try to use Windows 11 25H2 newest version (you can activate windows for free by using irm get.activated.win | iex on Windows PowerShell!).

• If it doesn't work, make a ticket on the website and send them your error code so they can look further into it.

• Otherwise, just try opening it again and again; the devs didn't give us any other info on it!

Page Infinite Load
When you try to access the dashboard and you are logged out (you can be logged out from time to time, that's normal), it will infinitely load. Access shikahub.org to go to the main page or click the top-left Shika logo to log back in.
Migrating Process
You get pings when you have the migration role; how it's assigned or removed is up to the devs. To start migration, click the button. If it says "already registered", you don't need to do anything else besides ensuring your shikahub account is linked to your Discord account. The subscription will be granted automatically.
Verification Code Not Received
When you don't get a code to verify your registration on shikahub.org, please make sure to check your Spam folder!
Autorisation Error: Empty Response from token endpoint
Delete cfg.ini located at:
%LOCALAPPDATA%/shika/launcher
HWID 不匹配错误 (HWID Mismatch)
请尝试直接在官方网站控制台重置您的 HWID。如果次数不够或失败，请在网站内提交工单处理。
启动器无法打开 (错误代码 0xc067 或类似错误)

• 建议升级并使用最新版本的 Windows 11 (25H2)。

• 如果系统未激活，可以打开 Windows PowerShell 并运行以下命令进行免费激活： irm get.activated.win | iex

• 如果问题依旧，请在官网提交工单并附上错误代码截图，以便开发团队协助排查。或者：尝试连续多次强制打开。

网页无限加载 (Page Infinite Load)
如果您的登录状态过期（自动登出），就会出现这种情况，属于正常现象。请直接访问主页 shikahub.org 或点击左上角的 Shika 图标重新登录。
账户迁移相关 (Migrating)
迁移身份组的分配完全由开发人员决定。点击网站上的迁移按钮，如果显示“Already Registered”（已注册），您只需要确保您的 Shikahub 账号已经成功绑定了 Discord 账号，剩余时间会自动完成同步转换。
未收到邮箱验证码
如果您在注册时没有收到验证码，请务必检查您邮箱的 垃圾邮件箱 (Spam) 或广告邮件箱。
授权错误 (Autorisation Error: Empty Response...)
请删除您电脑中残留的旧配置文件 cfg.ini，路径如下：
%LOCALAPPDATA%/shika/launcher
رسالة خطأ عدم تطابق HWID
جرّب إعادة تعيين HWID الخاص بك مباشرةً عبر لوحة تحكم الموقع. إذا انتهى الحد المسموح أو فشلت العملية, يرجى فتح تذكرة دعم في الموقع.
المشغل لا يفتح (رمز الخطأ 0xc067 أو ما شابه)

• يُنصح باستخدام أحدث إصدار من نظام التشغيل Windows 11 (25H2).

• إذا لم يكن الويندوز نشطاً، افتح Windows PowerShell وشغّل الأمر التالي لتنشيطه مجاناً: irm get.activated.win | iex

• إذا استمرت المشكلة، افتح تذكرة في الموقع وأرسل لقطة شاشة لرمز الخطأ لفحصه من قبل المطورين. أو: حاول فتحه مراراً وتكراراً.

الصفحة تستمر في التحميل دون توقف (Infinite Load)
هذا أمر طبيعي إذا انتهت جلسة تسجيل الدخول (تسجيل خروج تلقائي). الحل هو العودة للموقع الرئيسي shikahub.org أو الضغط على شعار Shika أعلى اليسار لتسجيل الدخول مجدداً.
عملية نقل الحساب (Migrating)
إعطاء رتبة النقل يعود للمطورين. اضغط على زر الهجرة في الموقع، وإذا ظهر لك "Already Registered"، فكل ما عليك فعله هو التأكد من ربط حسابك في Shikahub بحساب ديسكورد، وسيتم تحويل المدة المتبقية تلقائياً.
لم يصلني رمز تفعيل البريد الإلكتروني
إذا لم تصلك رسالة التفعيل عند التسجيل، يرجى التحقق من مجلد البريد المهمل (Spam) في إيميلك.
خطأ في التفويض (Empty Response from token endpoint)
احذف ملف الإعدادات القديم cfg.ini من المسار التالي في جهازك:
%LOCALAPPDATA%/shika/launcher
Ошибка HWID Mismatch
Попробуйте сбросить HWID прямо через панель на сайте. Если лимит исчерпан или выдает ошибку, создайте тикет поддержки на веб-панели.
Лаунчер не запускается (Ошибка 0xc067 или похожая)

• Рекомендуется использовать самую свежую версию Windows 11 (25H2).

• Если Windows не активирована, откройте Windows PowerShell и выполните команду для бесплатной активации: irm get.activated.win | iex

• Если лаунчер все равно вылетает, создайте тикет на сайте и прикрепите скриншот ошибки. В качестве альтернативы: попробуйте запускать лоадер несколько раз подряд.

Страница бесконечно загружается (Infinite Load)
Это нормально, если сессия авторизации истекла (автоматический разлогин). Для решения просто перейдите на главную страницу shikahub.org или нажмите на логотип Shika в левом верхнем углу, чтобы войти заново.
Процесс миграции аккаунта (Migrating)
Выдача роли миграции полностью контролируется разработчиками. Нажмите кнопку миграции на сайте; если написано «Already Registered», вам не нужно ничего делать, кроме как убедиться, что ваш аккаунт Shikahub связан с вашим Discord. Подписка перенесется автоматически.
Не приходит код верификации на почту
Если код не приходит при регистрации, обязательно проверьте папку Спам (Spam) или «Промоакции» в вашем почтовом ящике.
Ошибка авторизации (Empty Response from token endpoint)
Удалите устаревший файл конфигурации cfg.ini по следующему пути на вашем ПК:
%LOCALAPPDATA%/shika/launcher
