Installation Guide

  1. Registrasi Akun

    Buka & daftarkan akun baru Anda di shikahub.org

  2. Login

    Masuk menggunakan kredensial yang baru saja Anda buat.

  3. Aktivasi Kunci (Key)

    Input atau klaim kode lisensi yang sudah Anda beli di menu dashboard akun.

  4. Unduh Launcher

    Pergi ke halaman unduhan: shikahub.org/en/dashboard/downloads

    PENTING: Halaman unduhan tidak akan bisa diakses jika Anda melewati langkah registrasi dan aktivasi key di awal.
  5. Jalankan Loader

    Ekstrak file, klik kanan pada launcher, lalu pilih Run as administrator.

  6. Jalankan Shika

    Pilih dan eksekusi opsi Shika di dalam menu loader.

  7. Buka Game

    Jalankan game Anda. Sistem otomatis berjalan.

Launcher FAQ WIP

HWID Mismatch Message
Coba reset HWID Anda langsung via panel website. Jika limit habis atau gagal, silakan buat tiket aduan di menu dashboard web.
Launcher Won't Start (Error 0xc067 / Sejenisnya)

• Disarankan menggunakan Windows 11 versi terbaru (25H2).

• Jika belum aktif, buka Windows PowerShell lalu jalankan perintah: irm get.activated.win | iex untuk aktivasi gratis.

• Jika masih crash, buka tiket di web sertakan screenshot error code agar tim dev bisa cek manual. Alternatif lain: Coba buka paksa terus-menerus.

Page Infinite Load (Situs Muter Terus)
Hal ini normal terjadi jika session login habis (auto log-out). Solusinya, kembali ke homepage utama shikahub.org atau klik logo Shika di ujung kiri atas untuk login ulang.
Proses Migrasi Akun (Migrating)
Pemberian role migrasi sepenuhnya diatur dev. Klik tombol migrasi di web, jika tertulis "Already Registered", Anda cukup memastikan akun Shikahub sudah link dengan akun Discord Anda. Sisa sub otomatis terkonversi setelah sinkronisasi beres.
Kode Verifikasi Email Tidak Masuk
Pastikan untuk memeriksa folder Spam atau Promosi pada email yang Anda daftarkan.
Autorisation Error: Empty Response from token endpoint
Hapus file konfigurasi usang cfg.ini pada direktori komputer Anda berikut:
%LOCALAPPDATA%/shika/launcher
